### 关键观察

1. **Accept 规则是严格的 token_id 匹配**: `draft_token_id == argmax(target_logits)` 才接受，概率高低不影响规则本身
2. **Target 极度确定的位置** (pos=0, prob=0.999999): logit 差距 15.0，draft 只要猜对就必然接受
3. **Target 中等确定的位置** (pos=1,2, prob≈0.85): 存在竞争候选，但 top-1 明确，draft 猜对则接受
4. **Bonus token 不确定性高** (pos=3, prob=0.43): 三个候选 logit 差距 < 0.5，模型在此处有多种合理选择，这类位置是 draft model 最难预测的
5. **Draft 与 Target 对齐程度**: Step 1 top-1 与 target pos 2 top-1 完全一致，说明 draft model 在这个上下文中学到了正确的分布
